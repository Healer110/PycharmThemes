## 个人自用 Pycharm 主题分享

> 原因：jetBains 公司的 IDEA 似乎一直都不怎么重视主题配色这方面的元素，所以对于那些工具不怎么喜欢的起来。反而对于 Visual Studio Code 主题的情有独钟，比如 Sublime 里面很出名的 Monokai。直到后来我遇到了 Son of obsidian！

Pycharm 导入主题采用左上角菜单栏：File --> Import Seitings...
